# Integración Julia-Python

Incluye ejemplos con PyCall y PythonCall para importar módulos, llamar funciones y manipular datos entre ambos lenguajes.

- `pycall_basico.jl`: Uso de PyCall con math y numpy
- `pythoncall_basico.jl`: Uso de PythonCall con math y pandas

Requiere tener Python y los paquetes instalados en el sistema.
