# Plantilla para proyectos en Julia

## Estructura recomendada

- `/src` : Código fuente principal
- `/test` : Pruebas automáticas
- `/data` : Datos de entrada/salida
- `/notebooks` : Notebooks interactivos
- `/docs` : Documentación y tutoriales
- `/Project.toml` : Archivo de dependencias

## Consejos para organizar tu proyecto
- Usa módulos para separar funcionalidades
- Documenta cada función y archivo
- Escribe pruebas con Test.jl
- Mantén los datos y notebooks organizados
- Actualiza el archivo `Project.toml` con los paquetes usados

## Ejemplo de archivo Project.toml
```toml
[deps]
DataFrames = "a93c6f00-e57d-5684-b7b6-d8193f3e46c0"
Plots = "91a5bcdd-55d7-5caf-9e0b-520d859cae80"
Test = "eaa6bffe-5b3e-5381-b7d9-7c1b1f5f0a2b"
```

¡Usa esta plantilla para iniciar tus propios proyectos y mantener buenas prácticas desde el principio!
